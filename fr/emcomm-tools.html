EmComm-Tools OS

Explorer. Connecter. Répondre.

Une distribution Linux bilingue (français/anglais) basée sur Debian, conçue spécifiquement pour les communications d'urgence radioamateur. Démarrez depuis une clé USB, branchez votre radio, et vous êtes en ondes — avec JS8Call, Pat-Winlink, VARA, VarAC, fldigi, et YAAC avec plugin SAR prêts à l'emploi.

Description

EmComm-Tools OS Édition Debian est un système d'exploitation Linux prêt à déployer qui inclut tout ce dont un radioamateur a besoin pour les communications d'urgence numériques. Démarrez depuis une clé USB et vous êtes opérationnel.

Cette édition est construite sur Debian plutôt qu'Ubuntu, offrant des cycles de support plus longs, aucune restriction de marque sur la redistribution des ISO, et un support bilingue.

Fonctionnalités

  • Interface bilingue — français et anglais avec sélection de la langue au premier démarrage
  • Logiciels de modes numériques préinstallés : JS8Call, WSJTX, fldigi, Direwolf
  • Client Winlink (pat) avec VARA HF/FM préinstallé avec la permission du développeur
  • Wine32 préconfiguré pour les applications radioamateur Windows
  • Fonctionne hors ligne — tous les outils fonctionnent sans internet pour les scénarios SHTF
  • Images ISO prêtes à démarrer — aucun processus de compilation requis
  • Optimisé pour le déploiement terrain sur du matériel robuste (Panasonic Toughpads)
  • Basé sur Debian Stable avec cycle de support LTS de 3 à 5 ans
  • Base personnalisable pour les clubs, groupes régionaux et organismes d'urgence
  • Plugin YAAC pour les opérations SAR — Les équipes terrain peuvent marquer les évidences et objets trouvés lors des recherches directement sur la carte YAAC. Clic droit pour placer un marqueur en forme de losange avec catégorie, type et notes. Les objets peuvent optionnellement être diffusés via bulletin APRS
  • Open source sous licence Ms-PL — forkez-le, adaptez-le à vos besoins

Téléchargement ISO

L'ISO actuel (version 2.1.6) fait environ 4,7 Go et inclut tout ce dont vous avez besoin pour commencer. Les cartes ne sont pas incluses dans l'ISO afin de garder la taille du téléchargement raisonnable — au premier démarrage, le système vous proposera de télécharger et installer les tuiles cartographiques pour votre région.

Pour les appareils avec un stockage interne limité (comme les Panasonic Toughpads), les cartes peuvent être hébergées sur un disque externe ou une clé USB plutôt que sur le disque interne.

Nouveautés de la version 2.1.6

Répertoire de répéteurs (et-repeater)

Nouveau navigateur de répéteurs hors ligne pour trouver et programmer les répéteurs à proximité. Importez les exports CSV de RepeaterBook — aucune clé API requise. Pleinement fonctionnel sans internet après l'importation.

  • Import CSV — Exportez depuis repeaterbook.com, importez dans l'application. Les imports multiples fusionnent automatiquement
  • Géocodage automatique — Les emplacements sont géocodés via OpenStreetMap (Nominatim) lors de l'importation pour calculer les carrés de grille et les distances
  • Filtrage par distance — Filtrez les répéteurs par distance depuis votre station (utilise le carré de grille du profil utilisateur)
  • Filtres bande & mode — Filtrez par bande (10m/6m/2m/1.25m/70cm) et mode (FM/DMR/D-STAR/YSF/P25/NXDN)
  • Recherche — Recherchez par indicatif, fréquence, emplacement ou carré de grille
  • Colonnes triables — Cliquez sur n'importe quel en-tête de colonne pour trier en ordre croissant/décroissant
  • Favoris — Marquez des répéteurs pour créer une liste de favoris. Le filtre favoris a priorité sur tous les autres filtres
  • Édition & suppression — Modifiez les détails d'un répéteur (fréquence, offset, tonalité, emplacement, grille) ou supprimez des entrées
  • Programmation radio — Bouton « Set » en un clic pour programmer la fréquence, l'offset et la tonalité CTCSS via rigctld
  • Barre de progression — Progression du géocodage en temps réel lors de l'importation
  • Persistance — Données des répéteurs et favoris sauvegardés/restaurés via la persistance USB
  • Bilingue EN/FR

Coordonnées de la station

Champs optionnels de latitude/longitude dans l'assistant de premier démarrage et l'application de configuration utilisateur. Le carré de grille se remplit automatiquement à partir des coordonnées via la conversion Maidenhead. Utilisé par le Répertoire de répéteurs pour le calcul des distances et par le Digipeater APRS pour les balises de position.

VARA HF/FM et VarAC

VARA HF/FM — Préinstallé

Les modems VARA HF et VARA FM sont préinstallés dans l'ISO avec la permission écrite de José Alberto Nieto Ros (EA5HVK), le développeur.

Les utilisateurs doivent tout de même acheter leur propre licence directement auprès d'EA5HVK pour débloquer la vitesse maximale. La version préinstallée fonctionne en mode démo jusqu'à l'enregistrement. Visitez rosmodem.wordpress.com pour acheter votre licence.

Nous nous engageons à garder VARA à jour avec les nouvelles versions dès qu'elles sont disponibles.

VarAC — Préinstallé

VarAC est préinstallé dans l'ISO et prêt à utiliser dès le premier démarrage. Il fonctionne via Wine aux côtés de VARA HF/FM pour une solution de communications numériques complète, prête à l'emploi.

Outils de création USB bootable

Le seul logiciel dont vous avez besoin est un créateur de clé USB bootable. Voici les options les plus fiables :

Windows

  • Rufus — Rapide, fiable, open source. La référence pour Windows.
  • balenaEtcher — Processus simple en 3 étapes, valide l'écriture, multiplateforme.
  • Ventoy — Multi-boot — plusieurs ISO sur une seule clé USB.

macOS

  • balenaEtcher — Meilleur choix pour Mac — simple, fiable, application native.
  • UNetbootin — Multiplateforme, fonctionne bien pour les ISO Linux.
  • dd (Terminal) — Intégré, puissant mais demande de la prudence.

Linux

  • balenaEtcher — Même interface simple, disponible en AppImage.
  • Ventoy — Installez-le une fois, puis copiez simplement les fichiers ISO sur la clé.
  • GNOME Disks — Intégré à la plupart des bureaux GNOME, utilisez « Restaurer l'image disque ».
  • dd — Outil Unix classique, rapide et fiable.

Pour les débutants, balenaEtcher fonctionne sur toutes les plateformes et est l'option la plus simple. Pour les utilisateurs avancés, Ventoy vous permet de mettre EmComm-Tools, un installateur Windows et des outils de récupération sur une seule clé USB.

Pourquoi Debian ?

EmComm-Tools OS Édition Debian a débuté avec un simple fork du projet original EmComm-Tools OS Community par Gaston Gonzalez, TheTechPrepper (KT7RUN). Le projet original était construit sur Ubuntu avec la philosophie de créer un « appliance » figé dans le temps. Bien que cette approche ait du mérite, le paysage des logiciels radioamateur évolue rapidement — quand JS8Call, fldigi ou pat-winlink publient des correctifs, il faut pouvoir les appliquer. Un OS figé rend cela difficile voire impossible, et quand le système de base atteint sa fin de vie, votre gestionnaire de paquets cesse de fonctionner.

Ubuntu est en fait construit par-dessus Debian. Passer d'Ubuntu à Debian n'est pas un changement radical — c'est revenir à la source. Les principaux avantages :

  • Aucune restriction de marque — nous pouvons distribuer librement des images ISO précompilées
  • Cycles de publication plus longs et plus stables (3 à 5 ans LTS)
  • Accès direct aux vastes dépôts de paquets Debian
  • Debian Ham Radio Pure Blend — JS8Call, WSJTX, fldigi, Direwolf correctement empaquetés
  • Projet communautaire sans charge corporative
  • Maintenance active sans le risque d'un système figé et abandonné

L'esprit radioamateur

Ce projet n'est pas un produit fermé. C'est un point de départ — une base réutilisable et personnalisable que les clubs radio, groupes régionaux, organismes d'urgence et opérateurs individuels peuvent forker et adapter à leurs propres besoins. Partagez les connaissances, bâtissez la communauté.
