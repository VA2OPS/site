BBS Server

First Canadian hub station on The Packet Radio RF Forwarding Network (TPRFN). A 24/7 BBS node providing Winlink RMS, bulletin board, real-time chat, weather, and news — all over HF radio, no internet required.

What It Is

The VA2OPS BBS Server is a full-service packet radio node operating around the clock on HF. It serves as a hub station on the TPRFN network — a growing network of amateur radio stations that forward packet data exclusively over RF, without relying on internet (AXIP) links.

As the first Canadian hub on the network, VA2OPS extends TPRFN coverage into Canada, providing a relay point for store-and-forward messaging, keyboard-to-keyboard chat, weather bulletins, and news across the network.

Services

  • RMS Winlink — send and receive Winlink email over RF
  • BBS — classic bulletin board system for message posting and forwarding
  • Chat — real-time keyboard-to-keyboard communication
  • Weather (WX) — weather bulletins and reports
  • News — network news and announcements

About TPRFN

The Packet Radio RF Forwarding Network (TPRFN) is a network of amateur radio stations that connects packet radio systems using HF bands and modern digital modes like VARA HF. The network prioritizes Near Vertical Incidence Skywave (NVIS) links for reliable, RF-only communication — no internet backbone.

TPRFN provides a resilient digital communications infrastructure that works when traditional networks are unavailable. It supports store-and-forward messaging, interactive chat, intelligent routing, and digital file transmission between stations across the network.

How to Connect

To connect to the VA2OPS BBS, you need a VARA HF modem and a packet terminal client. Tune to the appropriate frequency for the time of day, and connect using your preferred packet software. For setup guides and details on joining the TPRFN network, visit the TPRFN website.
