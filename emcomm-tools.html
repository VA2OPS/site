EmComm-Tools OS

A bilingual (French/English) Debian-based Linux distribution built for amateur radio emergency communications. Fork of EmComm-Tools OS Community by TheTechPrepper (KT7RUN), maintained by Sylvain Deguire (VA2OPS).

What It Is

EmComm-Tools OS Debian Edition is a ready-to-deploy Linux operating system that comes pre-loaded with everything a ham radio operator needs for digital emergency communications. Boot from a USB drive and you're operational — no Linux experience required.

This edition is built on Debian rather than Ubuntu, providing longer support cycles, no trademark restrictions on ISO redistribution, and full bilingual French/English support — a necessity for Quebec's amateur radio clubs and emergency organizations operating under Loi 101.

Key Features

  • Bilingual interface — French and English with language selection at first boot
  • Pre-installed digital mode software: JS8Call, WSJTX, fldigi, Direwolf
  • Winlink client (pat) with VARA HF/FM pre-installed with developer permission
  • Wine32 pre-configured for Windows amateur radio applications
  • Offline-capable — all tools work without internet for SHTF scenarios
  • Ready-to-boot ISO images — no build process required
  • Optimized for field deployment on rugged hardware (Panasonic Toughpads)
  • Based on Debian Stable with 3–5 year LTS support cycle
  • Customizable foundation for clubs, regional groups, and emergency orgs
  • Open source under Ms-PL license — fork it, make it yours

Why Debian?

The original EmComm-Tools OS was built on Ubuntu, which has trademark restrictions that prevent redistribution of modified ISOs. By building on Debian, we can freely distribute ready-to-use ISO images. Debian also offers longer support cycles, direct access to the Debian Ham Radio Pure Blend package set, and no corporate overhead — just a solid, community-driven foundation.

Moving from Ubuntu to Debian isn't a radical change — it's going back to the source, since Ubuntu itself is built on top of Debian.

The Ham Radio Spirit

This project isn't a closed product. It's a vanilla starting point — a reusable, customizable foundation that radio clubs, regional groups, emergency organizations, and individual operators can fork and adapt to their own needs. Share the knowledge, build the community.
